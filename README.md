# CalculatorProject
## Java 과제 계산기 제작

### 목차
- 프로젝트 소개
- 주요 기능
- 프로젝트 구조
- 빌드/배포 방법


### 프로젝트 소개

- 해당 프로젝트는 Java 입문자로서 계산기를 제작해보며 Java에 적응하도록 돕는 과제로서 제작된 계산기 구현 프로젝트 이다.


### 주요 기능

- 사용자에게 두 개의 숫자(실수 포함)와 사칙 연산 기호(+, -, x, /)를 입력 받아 계산해주는 기능
- 위 과정에서 올바르지 않은 값이 입력되었을 때 경고문을 띄워주는 기능
- 해당 결과 값을 저장하는 기능
- 앞선 결과 값을 첫 번째 숫자로 상정하여 한 개의 숫자와 사칙 연산 기호를 입력 받아 새로운 계산을 이어서 해주는 기능
- 누적된 결과를 조회하는 기능
- 새로운 숫자를 입력 받아 저장된 결과 값들 중 해당 숫자와 비교하여 큰 결과들만 출력하는 기능
- 가장 먼저 저장된 결과를 삭제하는 기능


### 프로젝트 구조

- App : 
  - Main 구조를 담당하며 여러 값을 입력 받고 OperatorType에게 전달하여 계산을 수행하도록 지시하고, ArithmeticCalculator에게 해당하는 결과를 출력하라고 지시한다.
- ArithmeticCalculator : 
  - OperatorType에게 수행시킨 계산의 결과를 전달 받아 해당 결과들을 저장하고, App에게 전달한다.
- OperatorType : 
  - App에서 전달 받은 입력 값으로 계산을 수행하고, ArithmeticCalculator에게 반환한다.


### 빌드/배포 방법
- 학습용으로 제작 및 구현하였기에, 영리 목적을 제외한 모든 활동을 허용합니다.
